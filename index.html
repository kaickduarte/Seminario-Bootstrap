O que é o bootstrap?

Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web, usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo. Logo a baixo é possivel ver alguns dos diferentes tipos de buttons que podem ser estilizados de acordo com sua preferencia utilizando os codigos fontes do Bootstrap.

Origem

Originalmente desenvolvido para o site Twitter, chamado de Twitter Blueprint, desenvolvido por Mark Otto e Jacob Thornton como um instrumento para incentivar a consistência através de ferramentas internas.Antes de Bootstrap, várias bibliotecas foram utilizadas para o desenvolvimento de interfaces, gerando inconsistências e um elevado peso de manutenção.
